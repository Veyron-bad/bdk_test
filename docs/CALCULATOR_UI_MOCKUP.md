# Макет интерфейса калькулятора БДК

## Структура

```
┌─────────────────────────────────────────────────────────────────┐
│  Калькулятор стоимости БДК-балок                                │
│  Рассчитайте стоимость за несколько шагов                       │
├──────────────────────────────┬──────────────────────────────────┤
│  [1 Основные] [2 Дополн.] [3 Доставка]  │  ┌──────────────────┐ │
│                              │          │  │  БДК 6м×200×50×10│ │
│  Шаг 1:                      │          │  │  [Визуализация]  │ │
│  • Длина (м)      [6    ]    │          │  └──────────────────┘ │
│  • Высота (мм)    [200 ▼]    │          │  ┌──────────────────┐ │
│  • Толщина полки  [50  ▼]    │          │  │  Итого           │ │
│  • Толщина стенки [10  ▼]    │          │  │  125 000 ₽       │ │
│  • Порода         [Сосна▼]   │          │  │  Материал: ...   │ │
│  • Количество     [10   ]    │          │  │  Доставка: ...   │ │
│                              │          │  │  [Сохранить и    │ │
│  [Назад]           [Далее]   │          │  │   отправить]     │ │
└──────────────────────────────┴──────────┴──────────────────────┘
```

## Шаги

### Шаг 1 — Основные параметры
- Длина балки: number input, 1–12 м, шаг 0.5
- Высота сечения: select (200, 240, 300 мм)
- Толщина полки: select (40, 50, 60, 80 мм)
- Толщина стенки: select (8, 10, 12 мм)
- Порода древесины: select (Сосна, Лиственница, Кедр)
- Количество: number input, min 1

### Шаг 2 — Дополнительно
- ☐ Антисептик (+15%)
- ☐ Огнезащита (+25%)
- ☐ Фрезеровка пазов (800 ₽/м)
- ☐ Фрезеровка отверстий — при включении: поле «Кол-во на балку»

### Шаг 3 — Доставка
- Регион: select (Локальная, Региональная, Межрегиональная)
- При «Межрегиональная»: поле «Расстояние (км)»
- ☐ Срочная доставка

## Боковая панель (desktop)
- Визуализация балки (SVG)
- Итоговая сумма (крупно, оранжевый)
- Детализация: материал, фрезеровка, скидка, доставка
- Цена за шт
- Кнопка «Сохранить расчёт и отправить заявку»

## Адаптивность
- Desktop: форма слева, панель справа
- Tablet/Mobile: форма сверху, панель снизу (sticky снимается)
