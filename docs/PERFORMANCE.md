# Рекомендации по производительности

## Реализованные оптимизации

1. **Code splitting** — страницы загружаются по требованию (React.lazy)
2. **Lazy loading секций** — на главной ниже Hero подгружаются остальные блоки
3. **Lazy loading изображений** — `loading="lazy"` для картинок ниже сгиба
4. **Удалены неиспользуемые зависимости** — react-slick, slick-carousel (~50KB)
5. **Приоритет для LCP** — `fetchPriority="high"` для логотипа в Header

## Дополнительные рекомендации

### Изображения
- Сжимайте изображения (TinyPNG, Squoosh) перед добавлением в проект
- Используйте WebP/AVIF там, где поддерживается
- Указывайте `width` и `height` для предотвращения сдвига контента (CLS)

### Сборка
```bash
npm run build
```
Create React App минифицирует и разбивает бандл автоматически.

### Сеть
- Включите сжатие (gzip/brotli) на сервере
- Настройте кэширование статики (Cache-Control)
