# Концепция дизайна сайта производителя БДК-балок

## 1. Обзор

Сайт строительной компании, производящей балочно-диафрагмальные конструкции (БДК). Целевая аудитория: застройщики, инженеры-конструкторы, архитекторы и частные заказчики. 60% трафика — мобильные устройства.

---

## 2. Ключевые принципы

| Принцип | Реализация |
|---------|------------|
| **Профессионализм** | Чёткая структура, технические данные, сертификаты, ссылки на нормативы |
| **Надёжность** | Конкретные цифры, кейсы, фотографии производства и готовых объектов |
| **Доступность** | Простой язык, пояснения терминов, визуальные схемы, калькулятор |
| **Визуализация** | Крупные фото продукции, 360° или несколько ракурсов, галереи проектов |
| **Адаптивность** | Mobile-first, touch-friendly элементы, быстрая загрузка |

## 4. Moodboard — описание визуалов

### Фотографии
- Крупные планы балок: металл, сварные швы, маркировка
- Производственный цех: оборудование, персонал в СИЗ
- Готовые объекты: балки в здании, монтаж
- Схемы и чертежи на светлом фоне
- Сертификаты и документы

### Стиль изображений
- Естественное освещение, без избыточной обработки
- Чёткость и детализация
- Контраст: металл на нейтральном фоне

### UI-элементы
- Скруглённые углы 4–8px для кнопок и карточек
- Тени: лёгкие, `0 2px 8px rgba(0,0,0,0.08)`
- Иконки: линейный стиль, stroke 1.5–2px

## 6. Прототипы страниц

### 6.1 Главная страница

**Структура:**
- Hero (full-width слайдер)
- Блоки по 100% ширины, контент в контейнере max-width: 1200px
- Чередование фона: белый ↔ светло-серый
- Отступы секций: 80px (desktop), 48px (mobile)

**Ключевые точки:**
- Hero: min-height 400px (mobile), 500px (tablet), 600px (desktop)
- Крупные заголовки: 36–48px
- CTA всегда виден (sticky header или дублирование)

---

### 6.2 Каталог

**Макет:**
```
┌──────────┬─────────────────────────────────────────┐
│ ФИЛЬТРЫ  │  Сортировка: по цене / по размеру       │
│          │  ┌─────┐ ┌─────┐ ┌─────┐               │
│ • Тип    │  │Карт.│ │Карт.│ │Карт.│  (сетка 3col) │
│ • Длина  │  │  1  │ │  2  │ │  3  │               │
│ • Сечение│  └─────┘ └─────┘ └─────┘               │
│ [Сброс]  │  ┌─────┐ ┌─────┐ ┌─────┐               │
│          │  │  4  │ │  5  │ │  6  │               │
└──────────┴─────────────────────────────────────────┘
```

**Карточка товара (превью):**
- Фото (4:3 или 1:1)
- Название / размер
- Краткие хар-ки (длина, сечение)
- Цена (если применимо) или «По запросу»
- Кнопка «Подробнее»

**Мобильная версия:**
- Фильтры: выдвижная панель или accordion
- Сетка: 1 колонка
- Крупные tap-области

## 7. Гайд по типографике

### Шрифтовые пары

**Вариант A (универсальный):**
- Заголовки: **Montserrat** (Bold, SemiBold)
- Текст: **Open Sans** (Regular, Medium)

**Вариант B (технический):**
- Заголовки: **Exo 2** (SemiBold, Bold)
- Текст: **Source Sans 3** (Regular, Medium)

**Вариант C (классический):**
- Заголовки: **Roboto** (Bold, Medium)
- Текст: **Roboto** (Regular, Light)

### Рекомендуемый вариант: Montserrat + Open Sans

| Элемент | Шрифт | Размер | Вес | Межбукв. | Межстрочный |
|---------|-------|--------|-----|----------|-------------|
| H1 | Montserrat | 36–48px | 700 | -0.02em | 1.1 |
| H2 | Montserrat | 28–36px | 600 | -0.01em | 1.2 |
| H3 | Montserrat | 22–24px | 600 | 0 | 1.3 |
| H4 | Montserrat | 18–20px | 600 | 0 | 1.35 |
| Подзаголовок | Open Sans | 16–18px | 400 | 0 | 1.4 |
| Текст | Open Sans | 16px | 400 | 0 | 1.5 |
| Текст малый | Open Sans | 14px | 400 | 0 | 1.45 |
| Подпись/метка | Open Sans | 12px | 400 | 0 | 1.4 |
| Цифры/цены | Montserrat | 20–24px | 600 | 0 | 1.2 |
| Кнопки | Montserrat | 14–16px | 600 | 0.02em | 1 |

### Адаптивные размеры

| Breakpoint | H1 | H2 | Текст |
|------------|-----|-----|-------|
| 320–479px | 28px | 22px | 15px |
| 480–767px | 32px | 26px | 16px |
| 768–1199px | 40px | 30px | 16px |
| 1200px+ | 48px | 36px | 16px |

### Технические числа и единицы

- Цифры в таблицах: **tabular-nums** (выравнивание)
- Единицы измерения: через пробел, без точки (12 м, 300 кг)
- Большие числа: с разделителем тысяч (1 250 000 ₽)

### Google Fonts (импорт)

```css
@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&family=Open+Sans:wght@400;500;600&display=swap');

:root {
  --font-heading: 'Montserrat', sans-serif;
  --font-body: 'Open Sans', sans-serif;
}
```

## 8. Компоненты интерфейса

### Кнопки
- **Primary**: оранжевый фон, белый текст
- **Secondary**: синяя обводка, синий текст
- **Outline**: серая обводка, тёмный текст

### Карточки
- padding: 24px
- border-radius: 8px
- box-shadow: 0 2px 8px rgba(0,0,0,0.06)
- hover: лёгкое поднятие тени

### Поля ввода
- высота: 44px (touch-friendly)
- border: 1px solid #dfe6e9
- border-radius: 6px
- focus: обводка #2a5298

## 9. Адаптивность

- **Mobile-first** подход
- Breakpoints: 480, 768, 992, 1200px
- Touch targets: минимум 44×44px
- Изображения: srcset, lazy loading
- Мобильное меню: hamburger → full-screen или slide-in
