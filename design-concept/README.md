# Дизайн-концепция сайта БДК-балок

Набор материалов для визуальной и UX-концепции сайта производителя балочно-диафрагмальных конструкций.

## Содержимое

| Файл | Описание |
|------|----------|
| **DESIGN_CONCEPT.md** | Основной документ: принципы, палитра, структура главной, прототипы, компоненты |
| **TYPOGRAPHY_GUIDE.md** | Гайд по типографике: шрифты, размеры, адаптивность, CSS-переменные |
| **moodboard.html** | Интерактивный moodboard: цвета, типографика, UI-элементы, референсы контента |
| **wireframes.html** | Прототипы страниц: главная, каталог, карточка товара |

## Быстрый старт

Откройте в браузере:
- `moodboard.html` — визуальные референсы
- `wireframes.html` — схемы страниц (переключаются вкладками)

## Ключевые решения

- **Цвета:** синий (доверие), серый (техничность), оранжевый (CTA)
- **Шрифты:** Montserrat (заголовки) + Open Sans (текст)
- **Mobile-first:** 60% трафика с мобильных
- **CTA:** на каждом экране, приоритет — «Рассчитать» и «Заявка»
